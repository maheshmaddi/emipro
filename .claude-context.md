# EMIPro - Loan/EMI Calculator App

Read PLAN.md for the full design spec. This is Phase 1: Core build.

## What to build:

### Project Setup
- Next.js 14 (App Router) + TypeScript + Tailwind CSS + shadcn/ui
- Dark mode default with light mode toggle
- Color: Primary #6C5CE7, Secondary #00B894, Background #0F0F1A, Surface #1A1A2E, Card #16213E
- Fonts: Inter for text, Space Grotesk for numbers
- Zustand for state management
- Framer Motion for animations

### Core Components to Build:

1. **SliderInput** - Custom slider with:
   - Draggable range input
   - Editable number input (tap to type)
   - Quick-select chips below (common values)
   - Animated value label above thumb
   - Indian number formatting (lakhs/crores with ₹)

2. **LoanTypeSelector** - Horizontal scrollable pill tabs:
   - Home, Car, Personal, Education, Gold, Business
   - Each with emoji icon
   - Selecting pre-fills default values

3. **EmiHero** - The star of the show:
   - Massive EMI number (₹23,714/month)
   - Animated counter (odometer/digit-roll effect on change)
   - Three summary cards below: Principal, Interest, Total Payment
   - Subtle glow effect behind the number

4. **LoanConfig** - Three SliderInputs:
   - Loan Amount (₹5L - ₹5Cr, chips: 10L, 25L, 50L, 75L, 1Cr)
   - Interest Rate (4% - 24%, chips: 6, 7, 8, 9, 10, 12)
   - Tenure (1-30 years, chips: 5, 10, 15, 20, 25, 30)

5. **Charts** (Recharts):
   - Donut: Principal vs Interest split
   - Area: Outstanding balance over time

6. **Amortization Table**:
   - Month | EMI | Principal | Interest | Balance
   - Show first 12 months, "Show More" for full
   - Monthly/Yearly toggle
   - Download PDF & CSV buttons

7. **Layout**: Header (logo + dark/light toggle), Footer, responsive

### EMI Formula:
EMI = P × r × (1+r)^n / ((1+r)^n - 1)
where P=principal, r=monthly rate, n=months

### Key UX Rules:
- NO calculate button - EMI updates LIVE as sliders move
- EMI hero should be sticky on mobile
- Mobile-first responsive design
- All numbers in Indian format (lakhs/crores)
- Smooth animations on value changes (300ms)
- Min touch target 44px
